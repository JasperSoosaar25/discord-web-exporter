**⭐ Live Stars:** 0A lightweight, client-side web tool to export Discord chat history to HTML, JSON, and PDF directly from your browser. Inspired by the original DiscordChatExporter.

[🚀 Open Web App](https://JasperSoosaar25.github.io/discord-web-exporter/)

---

### ✨ Features
* **Privacy First:** Your token never leaves your browser. Requests are made directly to Discord's API.
* **Multiple Formats:** Export to **HTML** (clean layout), **JSON** (data), or **PDF** (via Print).
* **No Installation:** Run it directly in your browser without .NET runtimes.
* **Discord UI:** A clean, familiar dark-mode interface built with Tailwind CSS.

### 📖 How to Use
1.  Open the [Web App](https://JasperSoosaar25.github.io/discord-web-exporter/).
2.  Enter your **Discord Token**.
3.  Enter the **Channel ID** you want to export.
4.  Click **Export** and wait for the log to finish!
